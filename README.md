# CUSTOMER CHURN PREDICTION USING MACHINE LEARNING

## PROJECT OVERVIEW

Machine Learning project to predict telecom customer churn.

## TECHNOLOGIES USED

Python
Pandas
NumPy
Scikit-learn
Matplotlib
Seaborn

## MACHINE LEARNING MODEL

Random Forest Classifier

## DATASET

Telecom Customer Churn Dataset

## RESULT

Accuracy achieved: 78%

## KEY INSIGHTS

- Monthly Charges affect churn
- Total Charges influence customer retention
- Tenure impacts customer loyalty
